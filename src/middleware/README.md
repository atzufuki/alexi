# @alexi/middleware

CORS, logging, and error handling middleware for Alexi applications.

## Installation

```bash
deno add jsr:@alexi/middleware
```

## Usage

```typescript
import {
  corsMiddleware,
  errorHandlerMiddleware,
  loggingMiddleware,
} from "@alexi/middleware";
```
